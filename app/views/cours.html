b. Structure d'une page

HTML : ensemble de balises. <balise> bla bla </balise>

<head> </head> : en-têtes.
<body> </body> : corps/contenu de la page. Écrivez quelque chose.

Une balise peut avoir des options, appelées "attributs" : <balise option1="34" option2="Un peu de texte"> </balise>.
Comment lier des pages entre-elles ? La balise <a> permet de faire des liens : <a title="Lien vers Google" href="http://google.com">Le texte du lien</a>.

c. Les différentes balises de base

Mise en forme basique

  • <strong> </strong> : mettre en gras ;
  • <u> </u> : souligner ;

  • <h1> </h1> <h2> </h2> <h3> </h3> <h4> </h4> <h5> </h5> <h6> </h6> : titres (de très gros à très petit).

2. CSS

a. class/id

CSS : langage pour styliser les différents éléments/balises de la page. On peut modifier : couleur, couleur de fond, police, bordure (épaisseur, couleur), marges, position, taille...

Comment identifier un élément précis ?

  • On peut styliser toutes les balises du même nom :
    table {
    color: blue;
}
  • On ajoute un identifiant unique à une balise : l'attribut id.
    HTML : <td id="cellule-importante">. CSS : 
    #cellule-importante {
    border: 1px solid gray;
}
  • On veut modifier plusieurs éléments. On crée une class CSS et on l'ajoute aux éléments concernés.

    <table>
    <tr>
        <td class="cellule-peu-importante">Ligne 1, Colonne 1</td>
        <td class="cellule-peu-importante">Ligne 1, Colonne 2</td>
        <td>Ligne 1, Colonne 3</td>
    </tr>
    <tr>
        <td>Ligne 2, Colonne 1</td>
        <td>Ligne 2, Colonne 2</td>
        <td class="cellule-peu-importante">Ligne 2, Colonne 3</td>
    </tr>
</table>
    .cellule-peu-importante {
    opacity: 0.5;
}

Combinons les classes

Un élement HTML peut avoir plusieurs classes, et un id en plus : <img src="super-image.jpg" class="classe1 classe2 classe3" id="imageHeader" />.

<div> et <span>

Deux balises html sans signification.

<div> correspond à une boite.
<span> correspond à un morceau de ligne.

b. Les propriétés de base

Sur le texte

  • color: red|#AABBCC|rgb(234,8,23); : la couleur du texte ;
  • font-size: 34px; : la taille ;
  • font-family: "Helvetica", "Trebuchet MS", "Verdana"; la police (la première trouvée est choisie) ;
  • text-decoration: underline; : ajouter du soulignement ;
  • font-weight: 600; : épaisseur du texte (100 : fin, 300 : normal, 600 : gras).

Sur un bloc

  • margin-top: 4px; margin-bottom: 12px; margin-left: 0px; margin: 5px; : marges extérieures ;
  • background-color: gray|#333333|rgb(0, 34, 200); : couleur de fond ;
  • height: 300px; width: 90%; : dimensions. Unité : pixels, pourcentage de la largeur totale ;
  • text-align: center; : alignement du texte contenu dans le bloc ;
  • border-top: 4px solid yellow; border-left: 1px dotted blue; : bordure (épaisseur, style (pointillés ? normal ?), couleur).

b. La grille : cette idée géniale

Placement des différents blocs ? Menu à gauche ? Dimensions ? Marges entre les éléments ?

Bootstrap découpe la page en 12 colonnes. Tableau invisible pour organiser les élements.

<div class="container"> <!-- Toute la page dans un "container" -->
    <div class="row"> <!-- Je créé une ligne -->
        <div class="col-sm-4"> <!-- Ce bloc prend 4 colonnes -->
            Mon menu
        </div>
        <div class="col-sm-8"> <!-- Celui-ci prend les 8 suivantes -->
            Mon contenu
        </div>
    </div> <!-- J'ai 12 colonnes, je ferme ma ligne -->
    <!-- Je peux recommencer ici -->
</div>
